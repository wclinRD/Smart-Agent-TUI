ชุดโครงการชุมชนที่สร้างขึ้นบน Smart

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Smart ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-smart](https://github.com/awesome-smart/awesome-smart) และ [smart.cafe](https://smart.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
